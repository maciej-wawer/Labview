## Kamera cofania – LabVIEW + Arduino

Projekt przedstawia prototyp systemu wspomagania cofania z wykorzystaniem środowiska LabVIEW oraz mikrokontrolera Arduino, integrujący przetwarzanie obrazu z akwizycją danych z czujników w czasie rzeczywistym.

# Główne funkcjonalności

- Podgląd obrazu z kamery w aplikacji LabVIEW

- Pomiar odległości od przeszkód za pomocą czujnika

- Dynamiczna sygnalizacja dźwiękowa (buzzer) zależna od dystansu

- Komunikacja Arduino - LabVIEW

- Graficzny interfejs użytkownika (HMI)

# Technologie

- LabVIEW

- Arduino (C/C++)

- Czujnik odległości

- Moduł kamery
